## ⚙️ HOW TO DEPLOY ON VERCEL

```bash
1. Push this repo to GitHub
2. Go to https://vercel.com and click "Import Project"
3. Select your repo and set framework as "Other"
4. Set `index.html` as entry point
5. Deploy and GO LIVE 🚀
```
